Pet Clinic management system:
*Application:
+ The user will interact with the system via a command line interface where he/she can choose an option.
+ The desktop application will not have any visual friendly frames.

*Vet:
+ As a user I should be able to add a new vet. I should be albe to specify firstname/lastename/address/speciality.
+ As a user I should be able to view the details of the existing vet. I should be albe to view id/firstname/lastename/address/speciality.
+ As a user I should be able to update all existing vet by choosing an id. I should be able to edit firstname/lastename/address/speciality.
+ As a user I should be able to delete an existing vet by choosing the id.
+ As a user I should be able to view all vets. Ishould be able to view the following data id/firstname/lastname.

*Pet:
+ As a user I should be able to add a new pet. I should be albe to specify race/biethdate/isVaccinated/ownerName.
+ As a user I should be able to view the details of the existing pet. I should be albe to view id/race/biethdate/isVaccinated/ownerName.
+ As a user I should be able to update all existing pet by choosing an id. I should be able to edit race/biethdate/isVaccinated/ownerName.
+ As a user I should be able to delete an existing pet by choosing the id.
+ As a user I should be able to view all pets. Ishould be able to view the following data id/race/ownerName.

*Consult:
+ As a user I should be able to add a consult. I should be able to specify : vet id of an existing vet, pet id of an existing pet, date, description.
+ As a user I should be able to view all the consults. I should be able to view th following data : consult id, vet id of an existing vet, pet id of an existing pet, date, description.
+ As a user I should be able to update only the description for a specific consult by choosing the id.

*Import data:
+As a user I should be able to import all the data from a CSV file.

*Optional:
+As a user I should be able to authenticate before using the application.
